Agregar usuarios

Para agregar un nuevo usuario:
1. Crea el usuario en Firebase Console → Authentication → Add user
2. Copia el UID generado
3. Haz clic en "Agregar usuario" y asigna el rol

Solo el Owner puede agregar y gestionar usuarios.
